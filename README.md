# MDWS-JSON-to-Turtle
Very early version of a NodeJS script to convert the output of [MDWS-to-JSON](https://github.com/hetutrechtsarchief/MDWS-to-JSON) to a RDF Turtle file.

## Prerequisites
* NodeJS with NPM

## Install
```bash
npm install
```

## Usage
```bash
./MDWS-JSON-to-Turtle.js data/1001.json > data/1001.ttl
```
